About

React Seoul is a non-profit conference for React developers held by React Korea community.

React, which was first released in 2013 by Facebook, has been growing over years and was used in a lot of companies over the world including Korean ones. Although many developer conferences have React as a part of their proram, none of them have covered the vast world of React to its full extent. So, we organized React Seoul ourselves to boost its ecosystem in Korea.

Please fill out the Call for Proposals form here if you're interested in speaking at the first React conference in Korea. We would welcome any topics about React, React Native or the related technologies such as Babel, Next.js, Virtual DOM, Enzyme and Jest.

Whatever you want to know about the conference, fill free to ask on the #react-seoul channel in our slack.

This event is organized by community member volunteers.
