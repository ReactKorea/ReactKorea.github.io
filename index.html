행사 소개

React Seoul은 React 개발자를 위해 React Korea 커뮤니티에서 주관하는 비영리 컨퍼런스입니다.

페이스북이 2013년 공개한 React는 꾸준히 성장하여 전세계적으로는 물론 국내에도 많은 개발자와 기업이 React 기술을 도입하고 있습니다. 이 때문에 그간 많은 개발자 컨퍼런스에서 다루어졌지만 React의 넓고 다양한 세계를 담기에는 부족했습니다. 이에 React 생태계의 발전을 돕고자 한국에서는 처음으로 React 개발자 행사를 기획하게 되었습니다.

행사와 관련된 문의는 React Korea 슬랙에서 #react-seoul 채널을 이용해주세요.

본 행사는 커뮤니티 멤버들의 자발적인 참여로 이루어집니다.
